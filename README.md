# 2D Platformer Game

A 2D platformer developed using the Godot Engine and GDScript. The project demonstrates core platformer mechanics including player movement, jumping, sprite animation, collision detection, and physics-based interactions.

## Overview

This project serves as a foundation for a complete platformer experience and is being developed with scalability and maintainability in mind. The architecture is designed to support future gameplay features, additional levels, enemies, collectibles, and user interface systems.

## Features
* Character movement
* Jump mechanics
* Idle, running, and jumping animations
* Physics-based movement using CharacterBody2D
* Collision detection
* Tile-based terrain

## Technologies

* Godot Engine 4.x
* GDScript
* Git & GitHub

## Getting Started

### Prerequisites

* Godot Engine 4.x or later

### Installation

1. Clone the repository:

```bash
git clone https://github.com/wanjiru-noreen/2D-game-dev.git
```

2. Open Godot Engine.
3. Import the project folder.
4. Run the main scene.
